Email-Dedupe

Given a list of email addresses, one per line, this program will remove all duplicate emails while retaining original order

The output will be a downloadable .txt file containing one email address per line

REQUIREMENTS/ASSUMPTIONS:

  • One email address per line
  • Every line contains a single "@" (at) symbol that separates email name from domain name
